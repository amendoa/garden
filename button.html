Buttons

Standard buttons are used normally for form actions or can be used as hyperlinks. When using <button> element, always specify a type. If you care about accessibility, don’t forget to use role=button in <a> element.

Default buttons use these colors interface: .button-primary, .button-secondary, .button-neutral and .button-danger.

Using <button> element:

<button type="button" class="button"></button>

Using <a> element:

Default 
<a href="#" class="button"></a>

To disable a button, use disabled attribute or .button-disabled class simulate disabled prop.

Obs: when a button is disabled, button style will be the same for all types.

Using <button> element:

<button type="button" class="button" disabled></button>

Using <a> element:

Default 
<a href="#" role="button" class="button button-disabled"></a>
