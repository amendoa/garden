Scaffolding

The building blocks on how Garden works with CSS components.

Box-sizing

Garden applies box-sizing: border-box to all elements using the box-sizing global inherit:

html {
  box-sizing: border-box;
}

*,
*::before,
*::after {
  box-sizing: inherit;
}

Default styles

As in right now, Garden styles by default a couple of tags: h1-6, p, strong, b, em, i, u and a.

Normalize

In order to preserve a linear cross-browser behavior, Garden uses normalize.css.

Variables

Garden uses PostCSS Simple Vars plugins in order to work with variables.

Overriding default variables

The simple-vars plugin exposes a variables property, in order to inject variable defaults:

{
  "color-text": "#333333",
  "color-neutral": "#ffffff",
  "color-warning": "#fae800",
  "color-danger": "#ed1c24",

  "color-default": "#a6a6a6",

  "color-primary": "#72bf44",

  "color-secondary": "#bada2e",

  "default-font-size": "16px",
  "default-font-family": "Helvetica, Arial, sans-serif"
}

You can work providing a configuration file with your own defaults, or override during compilation time:

  /* your css */

  $color-primary: #ffffff;

  @import "garden/src/css/garden.css";
