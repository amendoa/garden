Images

Images by default, there isnt any styles. Garden provide some atoms to modify images, We have some shapes like ellipse or square and some styles as border.

Basic usage with .ellipse atom:

<img src="" class="ellipse" />

Using with .border and variations as .border-active and .border-disabled:

<img src="" class="ellipse border" />

.border-active:

<img src="" class="ellipse border border-active" />

.border-disabled:

<img src="" class="ellipse border border-disabled" />
