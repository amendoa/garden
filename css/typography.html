Typography

All typography tags/classes uses rem in order to set size. The $default-font-size variable uses 16px.

In case you use Garden css components individually (bundling with PostCSS), you can provide your own $default-font-size to change it. We provide both markup tags and classes in order to handle text styling with flexibility.

Defaults

Check out our scaffolding section for more info on providing your own defaults for typography.

Inline text

Tags such as <strong>, <b>, <em>, <i>, <u> are by default stylized. You can also use .strong and .italic, .underlined to reuse their styles.

italic
