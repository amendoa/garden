Panel

panel is a component responsible to display an image with a descriptive text, but in a more fashion way, compared to the thumbnail component:

Default Panel

The .panel component and itâ€™s variations stricly sets the height property, based on the panel variation.

Along with the .panel, the image should be used with .panel-image to properly position it, and with the .caption atom.

The width should be set by your current grid-system or hardcoded on your project:

Panel caption 
  <div class="panel">
    <img src="..." class="panel-image">
    <div class="caption">
      Panel caption
    </div>
  </div>

Large Panel

Panel caption 
  <div class="panel panel-large">
    <img src="..." class="panel-image">
    <div class="caption">
      Panel caption
    </div>
  </div>
:hover state

You can compose all .panel variations with a .panel-hover class, in order to achieve a simple :hover state, modifying the .caption behavior:

Panel caption with hover 
  <div class="panel panel-hover">
    <img src="..." class="panel-image">
    <div class="caption">
      Panel caption with hover
    </div>
  </div>
